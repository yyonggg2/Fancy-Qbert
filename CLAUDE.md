## Project Overview

Java remake of the classic Q*bert arcade game — CS final project. Grade depends on scope and quality.

Centerpiece feature: a pixel art canvas where the player draws their own character before playing, and that drawing becomes the actual in-game sprite.

GitHub repo: https://github.com/yyonggg2/Fancy-Qbert

## File Structure

- `Main.java` — creates the JFrame (window)
- `CanvasPanel.java` — creates the JPanel (drawable grid surface)
- `CONTEXT.md` — project notes
